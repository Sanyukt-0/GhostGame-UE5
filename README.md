# GhostGame — UE5.6 Stealth Puzzle Game

A top-down stealth puzzle game built in Unreal Engine 5.6 using C++.

## Systems Built
- Custom Player Controller (C++)
- WASD movement using Enhanced Input System
- Camera-relative directional movement
- Crouch system (in progress)

## Coming Soon
- Guard AI with patrol and line-of-sight detection
- Alert state machine (Patrol → Alert → Investigate)
- Puzzle elements (switches, doors)
- Win/Lose UI

## Tech Stack
- Unreal Engine 5.6
- C++
- Enhanced Input System
